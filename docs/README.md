# EO Activibase Documentation

Documentation for the EO (Epistemic Observability) Activibase project.

## Structure

### `/architecture`
Core EO theory and system architecture documentation.
- **EO_FRAMEWORK.md** - Core EO concepts and operators
- **EO_ARCHITECTURE_MANIFEST.md** - System architecture, module classification, and refactoring roadmap
- **EO_CAUSATION_THEORY.md** - EO-flavored theory of causation

### `/features`
Feature-specific documentation.
- **DATA_WORKBENCH_README.md** - Data workbench capabilities (views, structural operations, search)
- **FORMULA_FIELDS.md** - Formula field system API and usage
- **FORMULA_SYSTEM_GUIDE.md** - Formula system implementation guide for developers
- **LEAN_CONTEXT_GUIDE.md** - Memory-optimized context system

### `/guides`
Integration guides and how-to documentation.
- **EO_INTEGRATION_GUIDE.md** - How to integrate EO into applications
- **DATA_WORKBENCH_INTEGRATION.md** - Data workbench integration steps
- **EO_OPTIMIZATION_ROADMAP.md** - Codebase optimization plan with implementation status
- **MEMORY_OPTIMIZATION_GUIDE.md** - Memory optimization strategies and usage

### `/reference`
Design specifications and reference materials.
- **VIEW_FORMULAS_DESIGN.md** - View-specific formula system design document
- **connection_modes.md** - Connection modes dictionary reference
- **ui-refinement.md** - UI design system guidelines
